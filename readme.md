1.窗口可以完全后台,可以被遮挡,但是请不要最小化,否则将不能识图.

2.当使用高分辨率屏幕时,请不要勾选代替高dpi缩放行为(选择默认为空,如果勾选请取消)

3.您可以使用idea下载源码自行编译运行,也可以付费支持我们的发行版,该脚本没有比较难的技术问题,您付费购买的是我们的服务,感谢.
